Know Your NFT: What is metaData of NFT?



Newbie Explanation:



Metadata of NFT is a most important attribute. Now we know NFT is just a unique number which needs to be mapped to NFA. we can map it with the help of metadata. Metadata is the description about the NFA. For Example Metadata about Digital Art contains the URL of Art, Name, description, artist name. Metadata can be stored on blockchain ( On-chain ) or in private server ( Off-Chain )



Pro Explanation:



Metadata provides descriptive information for a specific token ID. In the case of the Crypto Kitty, the metadata is the name of the cat, the picture of the cat, a description, and any additional traits (called “cattributes”, in the case of CryptoKitties). In the case of an event ticket, the metadata might include the date of the event and the type of ticket, in addition to a name and description. Metadata can be stored on blockchain ( On-chain ) or in private server ( Off-Chain )
